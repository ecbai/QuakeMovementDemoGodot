A barebones recreation of Quake/Source Engine movement in Godot 4 (specifically made in 4.4.1).

#### DEFAULT CONTROLS:

- `WASD`: Move
- `Space`: Jump
- `LMB`: Fire
- `F11`: Fullscreen Toggle
- `R`: Reset Map
- `F1`: Decouple Mouse
- `ESC`: Close Game

Made using what I learned making Blast Processed, check it out here:
